# Nanolectronica-RF
Hands-on lab guides and course materials for microwave engineering students:
First one: Characterization of typical elements on a wafer. TLMs, Schottky and transistors.
Second One: Use of the open source software QUCS to extact SPICE model of a comercial SBD.
Third One: RF characterization of a transistor (Equivalent circuit, FT and FMAX) with open source software QUCS.
